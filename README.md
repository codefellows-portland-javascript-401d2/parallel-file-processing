### Instructions

#### To Test
- run "npm install" to install dependency modules
- run "npm test" to run a one-time test/lint process
- run "npm watch-js" to watch files for updates and run "test" again each time a JS file is saved (excludes node_module JS files)

#### To Run
- Using the command line, run "node index.js [enter name here, if more than one word put in quotes]"
